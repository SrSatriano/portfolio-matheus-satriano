# Portfólio — Matheus Rodrigues Satriano

Site pessoal em **Next.js 14** + **Tailwind CSS**, listando os 30 projetos open source do GitHub [@SrSatriano](https://github.com/SrSatriano).

## Executar localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build de produção

```bash
npm run build
npm start
```

## Deploy na Vercel

1. Crie um repositório `portfolio-matheus-satriano` no GitHub.
2. Importe o projeto em [vercel.com/new](https://vercel.com/new).
3. Framework: **Next.js** (detecção automática).

Ou via CLI:

```bash
npx vercel
```

## Personalizar

- Dados dos projetos: `data/projects.ts`
- Cores e fontes: `tailwind.config.ts` e `app/globals.css`
